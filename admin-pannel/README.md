# SD Fitness - Admin Panel

## Description
Professional admin panel for SD Fitness gym management system. Built with React, TypeScript, Vite, and shadcn/ui.

## Features
- 🎨 Professional glassmorphism UI with gradient accents
- 📊 Dashboard with KPIs and analytics
- 👥 Member management
- 💪 Trainer management
- 💳 Membership plans
- 📅 Class scheduling
- 💰 Payment management
- 📊 Analytics and reports
- ⚙️ Settings and configuration

## Development

```bash
# Install dependencies
npm install

# Run development server (port 3001)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
